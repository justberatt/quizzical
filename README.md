# quizzical

A simple quiz app built with React. Questions are fetched from the [Open Trivia Database](https://opentdb.com/) API.

## Features

- 5 random trivia questions per game
- Multiple choice answers
- Score tracking
- Animated start page transition — text fades out and slides left while decorative blobs scale and reposition before the questions page appears

## Tech Stack

- React
- Vite
- Open Trivia Database API
